About Square Finder App

The Square Finder App is a simple and user-friendly tool that allows you to quickly calculate the square of any number you input. It helps users practice basic math, verify answers, or perform quick calculations without a calculator. Built using HTML, CSS, and JavaScript, this project showcases how local storage and form handling work together to create a smooth user experience.

This app is part of a web development learning journey and was created to demonstrate DOM manipulation, form validation, and result sharing across pages.
